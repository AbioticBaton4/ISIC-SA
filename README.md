# Sistemas Abiertos (ESPARS3)

Este repositorio contiene el material, ejercicios y proyectos correspondientes a la materia de **Sistemas Abiertos**. El curso está diseñado para proporcionar las competencias teóricas y prácticas necesarias sobre la arquitectura, protocolos e interconectividad de redes bajo los modelos OSI y TCP/IP.

## Descripción del Curso
La asignatura de Sistemas Abiertos permite al estudiante analizar, comparar e implementar arquitecturas de red basadas en estándares abiertos, comprendiendo a profundidad la estratificación de protocolos, el enrutamiento, la interacción cliente-servidor y la seguridad en Internet.
